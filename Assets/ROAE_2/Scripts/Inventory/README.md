# Inventory

Contine logica pentru obiecte colectabile si inventar simplu.

Momentan include sistemul de shards. Daca inventarul devine mai mare, acesta este locul potrivit pentru:

- item models
- inventory state
- item pickup/give/use
- efecte legate de detinerea unui obiect

Prefabs asociate inventarului trebuie puse in `Assets/ROAE_2/Prefabs/Inventory`.
