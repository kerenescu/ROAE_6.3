# Dialogue

Contine baza sistemului de dialog: flags, triggers si efecte comune pentru alegerile din dialog.

Dialogurile concrete sunt de obicei `DialogueData` assets, nu cod. Codul din acest folder trebuie sa ramana generic si reutilizabil.

Pentru efecte specifice unui sistem:

- efecte generale de flags raman aici sau in `NarrativeV2/Shared`
- efecte pentru NPC-uri intra in `NPC_AI`
- efecte pentru Barista Welcome raman in `NarrativeV2/BaristaWelcome`
