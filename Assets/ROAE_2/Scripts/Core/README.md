# Core

Contine scripturi generale, mici si reutilizabile, care nu apartin unui sistem narativ sau unei scene anume.

Exemple:

- initializare de flags
- activare/dezactivare conditionata
- persistenta UI simpla
- blocari de control sau interactiune

Nu pune aici logica de NPC, dialog, puzzle sau UI complex. Daca un script creste si capata domeniu propriu, muta-l intr-un modul dedicat.
