# Presentation

Contine cod care afecteaza prezentarea vizuala sau senzatia scenei, fara sa decida progres narativ major.

Exemple:

- glow effects
- parallax
- typewriter text
- character scaling
- rotatii vizuale simple
- activare/dezactivare UI vizuala

Regula: daca scriptul schimba cum arata sau se simte scena, dar nu decide ce se intampla in poveste, probabil intra aici.
