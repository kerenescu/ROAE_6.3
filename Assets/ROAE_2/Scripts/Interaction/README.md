# Interaction

Contine scripturi pentru interactiunea directa cu lumea: hotspot-uri, click targets, trigger-e de obiecte, efecte de hover si schimbari simple declansate de player.

Acest modul nu decide continut narativ complex. El trebuie sa porneasca sisteme specializate, de exemplu dialog, puzzle sau schimbare de scena.

Exemple potrivite:

- hotspot label cleanup
- sparkle on hover
- door/usa triggers
- click spawner
- obiecte care apar/dispar pe baza unui flag
